# 상품 페이지 초안

## 상품명

개인투자자용 3계좌 통합 웹 포트폴리오 대시보드 구축

스윙매매 복기 / 장기투자 / 배당·커버드콜 관리 선택 가능

## 첫 문단

여러 증권사와 여러 계좌에 흩어진 투자 현황을 하나의 웹 대시보드로 정리해드립니다.

국내주식, 미국주식, ETF, 현금, 배당, 커버드콜 프리미엄, 스윙매매 기록을 고객님의 투자 방식에 맞춰 관리할 수 있습니다.

엑셀 파일을 단순 제공하는 상품이 아니라, 고객별 전용 링크로 제공되는 웹 포트폴리오 대시보드 구축 서비스입니다.

## 제공 기능

### 기본 대시보드

- 전체 평가금액
- 계좌별 평가금액
- 종목별 비중
- 자산군 비중
- 평가손익과 손익률
- 프라이버시 모드

### 스윙매매 복기 모듈

- 진입일/청산일
- 전략명
- 진입가/청산가
- 손익
- 보유기간
- 진입 사유
- 청산 사유
- 승률 및 손익비 확인

### 장기투자 모듈

- 목표 비중
- 현재 비중
- 괴리율
- 리밸런싱 점검용 표

### 배당·커버드콜 모듈

- 월별 배당금
- 옵션 프리미엄
- 세전/세후 현금흐름
- 배당+프리미엄 합산

### 데이터 관리

- 직접 입력
- CSV 업로드
- 브라우저 로컬 저장
- 백업 내보내기
- 백업 가져오기
- 전체 데이터 삭제

## 프라이버시 안내

증권사 계정 정보, 비밀번호, API 키는 요청하지 않습니다.

기본형에서는 고객님이 입력한 투자 데이터가 고객님의 브라우저 내부 저장소에 저장됩니다.

운영자 서버에 고객 금융 데이터를 별도로 저장하지 않는 구조로 설계합니다.

브라우저 데이터 삭제나 기기 변경에 대비해 백업 파일 내보내기와 가져오기 기능을 제공합니다.

## 패키지 예시

| 패키지 | 가격 | 포함 내용 |
|---|---:|---|
| Basic | 190,000원 | 고객별 링크, 기본 포트폴리오, 수동 입력, 로컬 저장 |
| Standard | 490,000원 | Basic + CSV 업로드 + 모듈 2개 + 백업/복원 |
| Premium | 990,000원 이상 | 전체 모듈 + 일부 커스텀 지표 + 사용법 문서 |
| Private Deploy | 1,500,000원 이상 | 고객 소유 GitHub/Vercel/Cloudflare 계정에 설치 |

## 포함 범위

- 기존 모듈 기반 화면 구성
- 고객별 링크 제공
- 수동 입력 화면 제공
- CSV 업로드 구조 제공
- 백업/복원 제공
- 일부 지표명 변경
- 표시 항목 일부 조정
- 모듈 선택
- 간단한 레이아웃 조정

## 미포함 또는 별도 견적

- 특정 종목 추천
- 매수·매도 신호
- 자동매매
- 증권사 자동 로그인
- 실시간 계좌 연동
- 신규 투자 전략 개발
- 완전 신규 앱 개발
- 고객별 복잡한 로직 개발
- 무제한 수정
- 투자 상담

## 면책 문구

본 서비스는 고객이 직접 입력한 투자 데이터를 정리하고 시각화하기 위한 웹 대시보드 구축 서비스입니다.

특정 종목의 매수·매도 추천, 투자 타이밍 제공, 수익률 보장, 손실 보전, 자동매매 신호 제공은 포함하지 않습니다.

투자 판단과 그 결과에 대한 책임은 고객 본인에게 있습니다.
