# 첫 고객 납품 체크리스트

## 판매 전

- [ ] 공개 데모 링크 준비
- [ ] 샘플 데이터만 포함되어 있는지 확인
- [ ] 실제 개인 투자 데이터 제거
- [ ] 프라이버시 안내 문구 확인
- [ ] 포함/미포함 범위 안내
- [ ] 가격표 안내
- [ ] 면책 문구 안내

## 고객 세팅

- [ ] 고객명 또는 프로젝트명 확정
- [ ] 선택 모듈 확정
- [ ] 고객별 `config.js` 제목 변경
- [ ] 고객별 `storageKey` 변경
- [ ] 고객별 사이트 비밀번호 변경
- [ ] 샘플 데이터 기본 로드 여부 결정
- [ ] 배포 플랫폼 결정

## 납품 전 테스트

- [ ] 직접 입력 테스트
- [ ] CSV 업로드 테스트
- [ ] 백업 내보내기 테스트
- [ ] 백업 가져오기 테스트
- [ ] 전체 데이터 삭제 테스트
- [ ] 프라이버시 모드 테스트
- [ ] 모바일 화면 확인
- [ ] 면책/프라이버시 화면 확인

## 납품 후 안내

- [ ] 고객에게 사이트 링크 전달
- [ ] 사이트 비밀번호 별도 전달
- [ ] 백업 파일 보관 방법 안내
- [ ] 브라우저 데이터 삭제 시 주의사항 안내
- [ ] 수정 가능 범위와 유지보수 조건 안내
- [ ] 후기 요청 시점 기록
