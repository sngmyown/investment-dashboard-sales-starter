# 개인투자자용 웹 포트폴리오 대시보드 판매형 스타터

이 패키지는 공개 데모와 고객 납품용 템플릿을 분리한 정적 웹앱입니다.

## 폴더 구조

- `public-demo/`: 실제 투자 데이터 없이 샘플 데이터로 보여주는 공개 데모
- `client-template/`: 고객별 링크로 납품하기 위한 템플릿
- `PRODUCT_PAGE_DRAFT.md`: 크몽/숨고/블로그 상품 페이지 초안
- `DELIVERY_CHECKLIST.md`: 첫 고객 납품 체크리스트
- `CSV_FORMATS.md`: CSV 업로드 형식

## 핵심 원칙

- 증권사 계정, 비밀번호, API 키를 요청하지 않습니다.
- 기본형 데이터는 고객 브라우저 `localStorage`에 저장됩니다.
- 운영자 서버에 고객 금융 데이터를 별도로 저장하지 않는 구조입니다.
- 백업 내보내기/가져오기 기능으로 고객이 직접 데이터를 관리합니다.
- 종목 추천, 매수·매도 신호, 자동매매, 수익률 보장 기능은 포함하지 않습니다.

## 실행 방법

각 폴더의 `index.html`을 브라우저에서 열면 실행됩니다.

```bash
# 예: 로컬 서버로 확인
cd public-demo
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.

## 고객 템플릿 비밀번호

`client-template/config.js`의 기본 비밀번호는 다음입니다.

```txt
change-me-2026
```

고객별 납품 전 반드시 변경하세요.

주의: 이 비밀번호는 정적 파일 안에 들어가는 가벼운 접근 제한입니다. 강한 보안 인증이 필요한 고객에게는 고객 소유 계정 배포, 이메일 기반 접근 제한, 정식 로그인 구조를 별도 견적으로 제안하세요.

## 배포 추천

초기에는 다음 중 하나로 충분합니다.

- Vercel
- Cloudflare Pages
- Netlify
- GitHub Pages
