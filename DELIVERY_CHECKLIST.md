# 납품 체크리스트

## 공개 데모 확인

- [ ] 실제 투자 데이터가 없음
- [ ] 샘플 데이터만 표시됨
- [ ] 스윙 계좌, 장기투자 계좌, 배당주 투자 계좌가 분리되어 보임
- [ ] 전체 현황에 통합 계좌 비중 도넛 차트가 보임
- [ ] 계좌별 3개 파이/도넛 차트가 보임
- [ ] 보유 종목 입력 폼에 계좌 선택 칸이 있음
- [ ] 매매 복기 입력 폼에 계좌 선택 칸이 있음
- [ ] 현금흐름 입력 폼에 계좌 선택 칸이 있음
- [ ] 프라이버시 모드 작동
- [ ] CSV 업로드 작동
- [ ] 백업 내보내기/가져오기 작동
- [ ] 모바일에서 화면이 깨지지 않음

## 고객 납품 전 확인

- [ ] `client-template/config.js`의 `title` 수정
- [ ] `storageKey`를 고객별 고유 값으로 수정
- [ ] `accessPassword` 수정
- [ ] 고객별 계좌명 필요 시 데이터 관리 화면에서 변경 안내
- [ ] 고객에게 CSV의 `accountId` 값 안내: `swing`, `long`, `dividend`
- [ ] 백업 내보내기 방법 안내
- [ ] 전체 데이터 삭제 방법 안내
- [ ] 포함/미포함 범위 안내
- [ ] 투자 조언이 아니라 기록·관리 도구임을 안내

## Premium 옵션

- [ ] Sunburst 차트는 기본형에 포함하지 않음
- [ ] Premium 또는 Private Deploy에서만 `enableSunburst: true`로 분리
- [ ] Sunburst는 “전체 계좌 → 계좌 내부 종목 비중” 고급 시각화로 설명

## 절대 하지 말 것

- [ ] 증권사 계정/비밀번호/API 키 요구 금지
- [ ] 종목 추천 금지
- [ ] 매수·매도 신호 제공 금지
- [ ] 수익률 보장 표현 금지
- [ ] 자동매매 기능 제공 금지
- [ ] 무제한 수정 약속 금지
