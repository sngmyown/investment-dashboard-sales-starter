# 상품 페이지 초안

## 상품명

개인투자자용 3계좌 통합 웹 포트폴리오 대시보드 구축

스윙매매 복기 / 장기투자 비중 / 배당·커버드콜 현금흐름 관리 선택 가능

## 첫 문단

여러 증권사와 여러 계좌에 흩어진 투자 현황을 하나의 웹 대시보드로 정리해드립니다.

스윙 계좌, 장기투자 계좌, 배당주 투자 계좌를 분리해서 입력하고, 전체 자산 비중과 계좌별 비중을 한 화면에서 확인할 수 있습니다.

본 서비스는 투자 조언이나 종목 추천이 아니라, 고객이 직접 입력한 투자 데이터를 정리하고 시각화하는 개인용 투자 관리 대시보드 구축 서비스입니다.

## 제공 기능

- 고객별 전용 웹 대시보드 링크
- 스윙 계좌 / 장기투자 계좌 / 배당주 투자 계좌 분리
- 전체 계좌 통합 현황
- 전체 계좌 비중 도넛 차트
- 계좌별 3개 파이/도넛 차트
- 보유 종목 직접 입력
- 매매 복기 입력
- 배당·커버드콜 현금흐름 입력
- CSV 업로드
- 백업 내보내기/가져오기
- 프라이버시 모드
- 전체 데이터 삭제

## Premium 옵션

Premium 또는 Private Deploy 패키지에서는 Sunburst 차트를 추가할 수 있습니다.

- 안쪽 원: 스윙 / 장기투자 / 배당주 계좌 비중
- 바깥쪽 원: 각 계좌 내부 종목 비중
- 전체 계좌 구조와 내부 구성 비중을 한 번에 확인

## 프라이버시 안내

증권사 계정 정보, 비밀번호, API 키는 요청하지 않습니다.

기본형에서는 고객님이 입력하거나 업로드한 투자 데이터가 고객님의 브라우저 내부 저장소에 저장됩니다.
운영자 서버에 고객 금융 데이터를 별도로 저장하지 않는 구조로 설계합니다.

브라우저 데이터 삭제나 기기 변경에 대비해 백업 파일 내보내기와 가져오기 기능을 제공합니다.

단, 고객님이 화면 캡처, CSV 파일, 백업 파일을 직접 전달하는 경우 해당 내용은 운영자가 확인할 수 있습니다.

## 패키지 예시

| 패키지 | 가격 | 포함 내용 |
|---|---:|---|
| Basic | 190,000원 | 고객별 링크, 3계좌 기본 포트폴리오, 수동 입력, 로컬 저장 |
| Standard | 490,000원 | Basic + CSV 업로드 + 백업/복원 + 모듈 2개 |
| Premium | 990,000원 이상 | 전체 모듈 + Sunburst 차트 + 일부 커스텀 지표 |
| Private Deploy | 1,500,000원 이상 | 고객 소유 GitHub/Vercel/Cloudflare 계정에 설치 |

## 포함 범위

- 기존 모듈 기반 화면 구성
- 고객별 링크 제공
- 수동 입력 화면 제공
- CSV 업로드 구조 제공
- 백업/복원 제공
- 계좌명 변경
- 일부 지표명 변경
- 표시 항목 일부 조정

## 미포함 또는 별도 견적

- 특정 종목 추천
- 매수·매도 신호
- 자동매매
- 증권사 자동 로그인
- 실시간 계좌 연동
- 신규 투자 전략 개발
- 완전 신규 앱 개발
- 고객별 복잡한 로직 개발
- 무제한 수정
- 투자 상담

## 면책 문구

본 서비스는 고객이 직접 입력한 투자 데이터를 정리하고 시각화하기 위한 웹 대시보드 구축 서비스입니다.
특정 종목의 매수·매도 추천, 투자 타이밍 제공, 수익률 보장, 손실 보전, 자동매매 신호 제공은 포함하지 않습니다.
투자 판단과 그 결과에 대한 책임은 고객 본인에게 있습니다.
